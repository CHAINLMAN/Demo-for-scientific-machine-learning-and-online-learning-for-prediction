# Demo-for-scientific-machine-learning-and-online-learning-for-prediction
This repository includes a Python implementation for scientific machine learning for solving PDEs with limited sensing and MATLAB code on online learning for prediction and control.
This repository provides a Python implementation of a scientific machine learning framework for solving PDEs under limited sensing. In this setting, a sequence of partial observations is used to reconstruct or predict the PDE solution through a physics-informed neural network. The implementation details and numerical examples are provided in the Jupyter notebook Random_sequential_initial.ipynb.
The repository also includes several open-source MATLAB projects on online learning for prediction and control. In particular, an optimization solver based on the online Newton method is implemented using the closed-form update derived in our previous work. Each MATLAB project can be run directly through the corresponding TestProgram.m script.
